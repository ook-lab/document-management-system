# Stage H: 構造化（チラシ商品情報抽出）

整形されたチラシテキストから、商品情報を構造化して抽出してください。

## 抽出する情報

1. **店舗情報**
   - 店舗名
   - チラシ期間

2. **商品情報**（各商品ごとに）
   - 商品名
   - 価格（数値）
   - 単位
   - カテゴリ
   - 備考（特価、産地など）

## 出力形式

```json
{
  "store_name": "XXXスーパー",
  "valid_period": {
    "start": "2025-03-15",
    "end": "2025-03-20"
  },
  "products": [
    {
      "name": "キャベツ",
      "price": 98,
      "unit": "1玉",
      "category": "野菜・果物",
      "notes": "国産"
    }
  ],
  "sale_info": [
    "ポイント5倍デー: 3/17",
    "数量限定商品あり"
  ]
}
```

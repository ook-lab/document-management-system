# sansu-drawing | 中学受験算数・数学 図形描画＆アニメーションアプリ

このアプリケーションは、中学受験算数や中学校数学で頻出する「図形の移動（転がり移動など）」や「回転体（3D回転）」を、自然言語（プロンプト）の指示からAIがPythonコード（Matplotlib + Numpy）を生成し、動的アニメーションとして描画・計算するStreamlitアプリです。

## 主な機能
1. **チャット指示（AIエージェント）**
   - 「1辺が6の正三角形の周りを、半径1の円が外側を滑らずに1周する軌跡を描いて」のような日本語の指示から、自動的に描画用Pythonコードを生成します。
   - 計算式（軌跡の面積や外周の長さ）も算出してLaTeXで表示します。
2. **対話的な修正**
   - 描画結果を確認後、「円の半径を2にして」「三角形の角を丸めて」「再生スピードを遅くして」などとチャットで指示を追加すると、前後の文脈を汲み取ってコードを修正・再描画します。
3. **コード直接編集**
   - 生成されたPythonコードを表示し、数値や色などをその場で直接書き換えて再実行・プレビューできます。
4. **定番パターンのプリセットボタン**
   - 初心者でもすぐに動かせるよう、「正三角形の周りを転がる円」「直角三角形の回転体」「おうぎ形の転がり移動」などのプリセットをワンクリックでロードできます。

## 起動方法

リポジトリルートディレクトリで以下のコマンドを実行します：

```bash
streamlit run services/sansu-drawing/app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。
※ `.env` ファイルに `GOOGLE_AI_API_KEY`（Gemini APIキー）が設定されている必要があります。
